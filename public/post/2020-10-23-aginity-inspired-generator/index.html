Load ‘em up

You can thank Gary Hutson for the below method of loading and or installing packages

install_or_load_pack <- function(pack){

  create.pkg <- pack[!(pack %in% installed.packages()[, "Package"])]

  if (length(create.pkg))

    install.packages(create.pkg, dependencies = TRUE)

  sapply(pack, require, character.only = TRUE)
  
}

packages <- c("randomNames","Hmisc")
install_or_load_pack(packages)

The Code block

name_generator <- function(selection_pool, final_selection) {
  col <- colours(distinct = TRUE)
  col <- gsub('[[:digit:]]+', '', col)
  col <- unique(col)
  col <- Hmisc::capitalize(col)
  my_name <-
    sub("\\s.*",
        "",
        randomNames::randomNames(
          n = length(col),
          name.sep = " "
        )
      )
  name_vector <-
    paste(sample(col, size = selection_pool),
          sample(my_name, size = selection_pool))
  print(sample(name_vector, final_selection))
}

Breaking it down

Using the inbuilt colours() function in R we extract colour names

col <- colours(distinct = TRUE)

Use regex to remove any numbers

col <- gsub('[[:digit:]]+', '', col)

As colours were differentiated by a number suffix, the removal of the number will create a number of identical entries.

So we drop those using unique()

col <- unique(col)

Lets capitalise the first letter of each of the elements in col

col <- Hmisc::capitalize(col)

Using randomNames we generate random names equal to the length of the col vector That gets wrapped in a sub() function that extracts all characters BEFORE the first space

my_name <-
    sub("\\s.*",
        "",
        randomNames::randomNames(
          n = length(col),
          name.sep = " "
        )
      )

Next we use sample() to select a pre-defined number of elements from both vectors and then again we use sample() to return a pre-defined number of names. The idea was to make it EXTRA random

  name_vector <-
    paste(sample(col, size = selection_pool),
          sample(my_name, size = selection_pool))
  print(sample(name_vector, final_selection))
